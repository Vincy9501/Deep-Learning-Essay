# 1. 安装labelimg

进入conda环境，cmd

`pip install labelimg -i https://pypi.tuna.tsinghua.edu.cn/simple`

`labelimg`

# 2. 标注文件

左边栏Open Dir打开数据集
Change Save Dir选择保存位置
\<Pascal> VOC 表示标注格式
Create RectBox 创建框，送开鼠标弹出标注内容。

# 3. 一些细节

View -> Auto save自动保存
View -> Discard 显示目标
单标签可以选择单一标签
Advanced Model -> 连续标注
ctrl + d 复制框
